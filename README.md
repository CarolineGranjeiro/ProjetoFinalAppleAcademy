# ProjetoFinalAppleAcademy

RPG: PERDIDO NA MITOLOGIA

Um jogo baseado na história de um jovem acorda sem saber seu nome, idade, sua prórpria identidade! Ele recebe a chance de ter seu destino mudado pelas Cartas de Delfos. 

Características:  lutar contra  monstros mitológicose achar o portal para o Monte Olimpo

RPG: A forja do engenheiro mágico. Um jogo de RPG, desenvolvido em Swift como projeto para o Apple Developer Academy.

Como Jogar Comandos do Jogo

Durante o jogo, você usará números para fazer escolhas.

Aula 1 - Introduction Variáveis (var) e constantes (let) Tipos de dados (String, Int, Bool, Dictionary) String interpolation Optionals Type safety e type inference readLine() para input do usuário Operadores aritméticos e de comparação

Aula 2 - Control Estruturas condicionais (if/else) e Operadores lógicos

Aula 3 - Collections Arrays Dictionaries Loops (for, while) Range operators

Aula 4 - Functions Funções com parâmetros e retorno .
